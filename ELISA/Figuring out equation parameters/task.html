Description

As you already know, the next step in obtaining sample concentrations is curve fitting. In this stage, calculate all parameters of the four parametric logistic curve (4PL) equation using data from standard_values.csv: \[y = D + {{A - D} \over 1 + {x \over C}^B}\]

Objectives
  1. Take the path to the CSV file with the standard values using input() function. Then process the file using pandas.read_csv();
  2. Find an average optical density (similar to the procedure in Stage1) and name the column AverageOD;
  3. Write a 4PL equation as a function in Python;
  4. With the scipy.curve_fit() Python function, find all equation parameters, using the standard values data, and print the result (exactly as shown below).

For example, it can look like this:

from scipy.optimize import curve_fit

def func(x, a, b, c):

    return a * np.exp(-b * x) + c

popt, pcov = curve_fit(func, xdata, ydata) 

print(popt)

Here, xdata equals known concentrations for the standard samples, a column of your DataFrame (an array), and ydata is the known optical density (OD) values for standard samples, a column "AverageOD" of your DataFrame (an array).

The function in the example is NOT RELATED to the 4PL equation; create the function, similar to the one in the example.

Example

Input: CSV file with standard values

Output (the value of popt):

array([  0.24379838,   1.68910347, 116.67136174,   2.86685359])

Please keep in mind that the output should be given as an array, not a list; thus being said, simply printing a variable is enough, no need to change the type of a variable.
