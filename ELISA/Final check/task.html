Description

In ELISA experiments, researchers use sample dilutions to prevent any potential assay interference and to get more accurate results. How to be sure that the protein is still detectable? For that, you can calculate the LOD, the lower limit of detection.

Your colleague ran a series of experiments to find the LOD and kindly shared it with you. Try to upgrade your function so it would tell you (based on the earlier obtained concentrations) if you can still detect the protein in the sample.

Objectives

Upgrade the elisa_automation function, so the data frame would have a new column called Protein detectable with True/False values.


\[LOD value = 1 ng/ml\]

Example

Input:

  1. File in CSV format with raw signals;
  2. File in CSV fromat with standard values.
