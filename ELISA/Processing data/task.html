Description

The laboratory, that has sent you the data, studies the role of a certain protein in the disease \(X\). You have received two files: a file with raw signals (OD), elisa_data.csv, and a file with standard values of a studied protein, standard_values.csv.

If the received files are corrupted, you can download them manually: elisa_data.csv and standard_values.csv.

Objectives
  1. Take the path to the CSV file with raw signals using input() function. Then process the file using pandas.read_csv() and examine the columns in the file;
  2. Create two columns – AverageOD and CorrectedOD – and fill them up with values, so you could proceed with the calculations!
  3. Save the updated file as elisa_result.csv.

Just a reminder, for AverageOD, find an average of duplicates (divide the sum of all observations (OD1, OD2) by the number of observations):

\[AverageOD = {{OD1 + OD2} \over 2}\]

For CorrectedOD: subtract the blank values (subtract a calculated average blank value from all other average values in the table):

\[CorrectedOD = {Average - BlankAverage}\]

Of course, it is simpler to pass the file directly to the pandas.read_csv() function. However, using the input() function provides greater flexibility for the script; instead of replacing the file path every time you use it, you can provide it later at runtime.
Example

Content of CSV file with raw signals:

OD1 OD2
Sample 1 1.253 1.269
Blank 0.172 0.175

Output: elisa_result.csv file with the following structure:

OD1 OD2 AverageOD CorrectedOD
Sample 1 1.253 1.269 1.261 1.087
Blank 0.172 0.175 0.174 0
